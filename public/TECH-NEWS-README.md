# Tech News Daily - Final Output

This is the complete, working example from the HTML and CSS lessons.

## Files Created

1. **tech-news.html** - The complete HTML structure (Version 5 from HTML lesson)
2. **tech-news.css** - The complete CSS styling (Version 7 from CSS lesson)

## How to View

1. Open `tech-news.html` in your web browser
2. The page will automatically load the CSS from `tech-news.css`

## What You'll See

A fully styled, responsive news website with:

### Header

- Dark background (#1a1a1a) with red accent (#c00)
- "Tech News Daily" title
- Sticky navigation that stays at top when scrolling
- Hover effects on navigation links

### Main Content

- Featured article about AI with:
  - Large headline
  - Author and date information
  - Responsive image with hover zoom effect
  - Multiple paragraphs
  - Sections with "Key Benefits" list
  - Article footer with tags

### Related Stories

- Two article cards that:
  - Lift up on hover
  - Show shadow effects
  - Display in a grid layout

### Sidebar

- Sticky positioning (follows you as you scroll)
- Trending topics list with custom bullets
- Newsletter signup form with:
  - Styled input field
  - Focus effects (red border when clicked)
  - Styled submit button with hover effect

### Footer

- Dark background matching header
- About section
- Contact information
- Copyright notice

## Responsive Design

The site adapts to different screen sizes:

- **Desktop (1200px+)**: Full layout with sidebar
- **Tablet (768px-1199px)**: Adjusted spacing
- **Mobile (below 768px)**:
  - Single column layout
  - Stacked navigation
  - Sidebar moves below main content
  - Smaller text sizes

## CSS Features Demonstrated

1. **Box Model**: Padding, margin, borders
2. **Positioning**: Sticky header and sidebar
3. **Flexbox**: Navigation layout
4. **CSS Grid**: Page structure and article grid
5. **Transitions**: Smooth hover effects
6. **Transform**: Image zoom, button lift
7. **Box Shadow**: Card depth effects
8. **Media Queries**: Responsive breakpoints
9. **Pseudo-classes**: :hover, :focus, :before
10. **Typography**: Font families, sizes, line heights

## Color Scheme

- **Primary Red**: #c00 (links, accents, borders)
- **Dark Red**: #900 (hover states)
- **Dark Gray**: #1a1a1a (header, footer)
- **Text Gray**: #333 (body text)
- **Light Gray**: #666, #999 (meta text)
- **Background**: #f9f9f9 (page background)
- **White**: #fff (content cards)

## Typography

- **Headings**: Georgia, Times New Roman (serif)
- **Body**: Georgia, Times New Roman (serif)
- **Form Inputs**: Arial (sans-serif)

## Try These Interactions

1. **Hover over navigation links** - They turn red with background
2. **Hover over the main image** - It zooms in slightly
3. **Hover over related article cards** - They lift up with shadow
4. **Click in the email input** - Red border appears
5. **Hover over the subscribe button** - It darkens and lifts
6. **Scroll down** - Header stays at top (sticky)
7. **Resize browser window** - Layout adapts responsively

## Learning Points

This example demonstrates:

- Separation of concerns (HTML structure, CSS presentation)
- Semantic HTML elements
- Modern CSS layout techniques (Flexbox, Grid)
- Responsive design principles
- User experience enhancements (hover effects, transitions)
- Accessibility considerations (semantic elements, labels)

To customize this:

4. Adjust spacing (modify padding and margin values)
